# AVCO Pizza Co. Example Site

Open `index.html` in a browser to review the example pizzeria site.

This folder contains a static restaurant site that can be adapted for another pizzeria or casual dining brand. It includes:

- sticky navigation
- full-bleed food hero
- order-first calls to action
- featured menu scroller
- alternating image and copy sections
- catering/event promo blocks
- guest reviews
- FAQ accordions
- location tabs
- mobile sticky order bar

To customize it, start with the CSS variables at the top of `styles.css`, then replace the brand name, images, menu items, reviews, locations, and CTA URLs in `index.html`.
